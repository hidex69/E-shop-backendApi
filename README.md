# E-shop-backendApi
<h1> Shop </h1>
REST API with JWT-security to access a database with products to order them

<h2>API includes:</h2>

If **guest**:
 - get all products
 - get one product
 - get login/register page

If **user**:
 - guest capabilities
 - ordering
 - logout action
 - add/remove product to/from basket

If **admin**:
 - user capabilities
 - product/user management
 - logout action
